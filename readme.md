## 头部代码

> 原小腿代码，继承框架代码，修改而成。

### 修改

#### 框架问题
- CAN1 和 顶板的 CAN3 打开了自动重发和busoff，解决了一定时间（20s）左右会发送不出数据的问题。、

- DM 电机 can_send 没有初始化变量为空，导致CAN发送出现问题。

- 键鼠存在形参调用，左右键错误等。


### CAN 设备

| 电机 | ID | master ID | freq | CAN|
| -- | -- | -- | -- | -- |
| yaw | 0x01 | 0x11 | 500hz | CAN1 |
| pitch | 0x02 | 0x12 | 500hz | CAN1 |
| F42H7  | 0x1C | - | 500hz | CAN1 |
| boost_G | 0X203 | 0X200 | 1khz | CAN1 |
| boost_left | 0x201 | 0x200 | 1khz | CAN2 |
| boost_right | 0x202 | 0x200 | 1khz | CAN2 |
